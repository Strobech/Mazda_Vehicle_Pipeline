# Mazda Vehicle Pipeline

The Mazda Vehicle Pipeline is a Python framework for building validated vehicle manifests from Mazda Maya scenes, MEL visibility scripts, and official Mazda order forms.

## Project Goals

- Produce a single Vehicle Manifest as the source of truth.
- Support multiple Mazda vehicle programs.
- Generate data for Cinema 4D.
- Validate all engineering data.
- Be fully testable.

- CX-50 MY26
